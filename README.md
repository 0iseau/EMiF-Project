# EMiF-Project

## Predicting U.S. Financial Markets Using Oil Price Increase: The Role of Macroeconomic Conditions and Shock Types

## Reproducible setup (so notebooks work for everyone)

### Option A (recommended): `uv` + local venv

- Create venv: `uv venv .venv`
- Install deps: `uv pip install -r requirements.txt`
- In VS Code, select the Python interpreter `.venv` as the notebook kernel.

### Option B: pip + venv

- Create venv: `python -m venv .venv`
- Install deps: `.venv\\Scripts\\python -m pip install -r requirements.txt`
- Select `.venv` as the notebook kernel.
